Transform...

This menu item is used when you want to rotate/translate the mesh.

Usage

  1. Select a mesh in the mesh tree.





  2. Use the Menu "File > Transform...".
  3. Write the rotations and translations. The rotations follow the order X > Y > Z.
  4. Press "OK" or "Enter" if you want to apply the rotations/translations, press "Align Z with tool" if you want to correct the Z alignment of the mesh or press "Cancel" to abort.





  5. If you selected "Align Z with tool", you need to create a vector that points to the desired Z, select the first vector point. It is easier if you use a building as reference.





  6. Select the last point.





  7. The mesh will be rotated in a way that the vector is equal to the Z+ axis.
