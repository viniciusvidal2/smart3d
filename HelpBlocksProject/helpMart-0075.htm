Fast quadric simplification...

This menu item is used to reduce the number of faces in a mesh (does not work with textured meshes).

Usage

  1. Select a mesh in the mesh tree.





  2. Start the tool using the Menu "Filters > Fast quadric simplification...".
  3. Choose the percentage of reduction you want.





  4. Press "OK" to start the simplification or "Cancel" to abort the operation.
