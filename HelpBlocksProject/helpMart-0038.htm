Poisson...

This menu item is used to reconstruct a mesh using poisson.

Usage

  1. Select a mesh in the mesh tree.





  2. Start the tool using the Menu "Reconstruction > Poisson...".





  3. Define the parameters, or press "Default" to load the default parameters.
    • Depth - The bigger the depth the best the result, but you will consume more RAM and the time will increase significantly.
    • Boundary
      • Free - No boundary constraints.
      • Dirichlet - Dirichlet boundary constraints (The surface will be closed).
      • Neumann - Neumann boundary constraints.
  4. Press "OK" to start the reconstruction or "Cancel" to abort the operation.
  5. When it finishes, the result will appear in the mesh tree as "Poisson Reconstruction (Depth < the depth you chose >)".
