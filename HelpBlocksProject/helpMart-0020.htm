Angle Tool

This tool is used to measure the angle formed by three 3D points.

Usage

  1. Select a mesh in the mesh tree.





  2. Press the angle icon in the toolbar.





  3. Click in the first point using the right mouse button.





  4. Click in the second point using the right mouse button.





  5. Click in the third point using the right mouse button.





  6. The angle will appear in the screen.
  7. You can move the points by going on top of them, holding the right mouse button and moving the mouse.
  8. If you want to make another measure, just press "Ctrl + Z".

Shortcuts

  • Ctrl + A - Enable/disable the angle tool.
  • Ctrl + Z - Remove the last added point/Restart the tool.
