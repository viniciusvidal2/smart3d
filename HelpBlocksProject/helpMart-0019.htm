Measure Tool

This tool is used to measure the distance between two 3D points.

Usage

  1. Select a mesh in the mesh tree.





  2. Press the measure icon in the toolbar.





  3. Click in the first point using the right mouse button.





  4. Click in the second point using the right mouse button.





  5. The measure will appear in the screen, inside the parentheses are the delta X, Y and Z. If the measure does not have a measure unit, you can to use the calibration tool to calibrate the mesh.
  6. You can move the points by going on top of them, holding the right mouse button and moving the mouse.
  7. If you want to make another measure, just press "Ctrl + Z".

Shortcuts

  • Ctrl + M - Enable/disable the measure tool.
  • Ctrl + Z - Remove the last added point/Restart the tool.
