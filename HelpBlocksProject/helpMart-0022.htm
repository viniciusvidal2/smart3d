Elevation Tool

This tool is used to create an elevation map and an elevation profile.

Usage

  1. Select a mesh in the mesh tree.





  2. Press the elevation icon in the toolbar.





  3. Will be generated an elevation map of the mesh. You can use the slider to change the zero, or if you press "Ctrl + Y" and click with the middle mouse button in the point you want the zero to be.





  4. To find the elevation of a point just click with the middle mouse button.





  5. To define an elevation profile click with the right mouse button to define the frist point.





  6. Click with the right mouse button to define the second point.





  7. The elevation profile will appear as a plot in the screen and represented as a white line in the visualization area. The plot is oriented in the +X axis direction, so the first point will always be the point with the smaller X coordinate. If the elevation does not have a measure unit, you can to use the calibration tool to calibrate the mesh.
  8. You can move the profile line points by going on top of them, holding the right mouse button and moving the mouse.
  9. If you want to make another profile, just press "Ctrl + Z".

Shortcuts

  • Ctrl + E - Enable/disable the elevation tool.
  • Ctrl + Y - Enable/disable the mouse definition of zero.
  • Ctrl + Z - Remove the last added point/Restart the profile line.
