Delete Points Tool

This tool is used to delete points of a point cloud. This tool only works with point clouds.

Usage

  1. Select the item "Tool delete points" in the VR menu.





  2. A selection sphere will be placed in the right controller.





  3. To select the points, place the points you want to select inside the sphere and press the trigger.





  4. The selected points will be painted in red.





  5. If you want to cancel the selection press the trigger with no points inside the sphere.
  6. To delete the selected points press the grip button.
  7. The points will be deleted.





Shortcuts

  • Trigger - Select the points inside the selection sphere.
  • Grip button - Delete the selected points.
