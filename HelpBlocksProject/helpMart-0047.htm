Change the focus point

This function is useful if you need to focus in some specific point.

Usage

  1. Align the point you want to focus with the center of the visualization area.
  2. Press "Ctrl + U". The program will shoot a ray trace and will change the focus to the first valid point.
  3. The focus will be changed if some valid point was found.

Shortcuts

  • Ctrl + U - Set the focus point.
