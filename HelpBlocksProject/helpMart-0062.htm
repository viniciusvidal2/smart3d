Send to VR Tool

This tool is used to send a mesh to the VR oculus.

Usage

  1. Make sure your VR oculus is working.
  2. Select a mesh in the mesh tree.





  3. Press the Send to VR icon in the toolbar.





  4. Wait until this window appear.





  5. Enjoy the VR experience.
  6. If you want to exit the VR press the "Stop VR" button.
  7. It will ask you if you want to keep the changes made in VR, press "Sim" to keep them or "Não" to discard.
