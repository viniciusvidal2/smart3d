Elevation Tool

This tool is used to create an elevation map.

Usage

  1. Select the item "Tool elevation" in the VR menu.





  2. Will be generated an elevation map of the mesh.





  3. To find the elevation of a point, press the trigger button and point the controller to the point you want to know the elevation, and release the trigger.





  4. You can change the zero definition by pressing the grip button, then press the trigger button and point the controller to the point you want to define as zero, and release the trigger.





Shortcuts

  • Trigger - Make a picking action.
  • Grip button - Enable/disable the definition of zero.
