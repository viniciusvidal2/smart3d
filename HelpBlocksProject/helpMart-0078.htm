Reconstruct clusters...

This menu item is used to reconstruct a group of clusters. To create the mesh for each cluster the poisson reconstruction will be used, and the result will be texturized.

Usage

  1. Start the tool using the Menu "Reconstruction > Reconstruct clusters...".
  2. Select a folder that contains the clusters and optionally the .nvm file.
  3. Press "OK" to start the reconstruction or "Cancel" to abort the operation.
  4. When it finishes, it will ask if you want to see the result, press "Yes" to load the result or "No" to dismiss it.
